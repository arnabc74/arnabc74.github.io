C Programming আমার C Programming বইয়ের webpage এটা৷ চাররকম জিনিস পাবে এখানে--
  • এক, যে সব প্রোগ্রাম দেখানো আছে বইটাতে, সেগুলো ডাউনলোড করতে পারবে এখান থেকে
  • দুই, বইয়ের কয়েক জায়গা থেকে বিভিন্ন ভিডিওর QR link দেওয়া আছে৷ সেগুলোও পাবে নীচে৷
  • তিন, যারা Android দিয়ে C শিখতে চাও, তারা চাইলে এখান থেকে ডাউনলোড করতে পারো৷ তবে প্রথমে Google playstore থেকেই install করার চেষ্টা কোরো৷ নেহাত যদি তাতে অসুবিধা হয়, তবেই খালি এখান থেকে ডাউনলোড করো৷ এখান থেকে তুমি একটা apk ফাইল পাবে৷ সেটা তোমার মোবাইলে রেখে tap করলে install হবার কথা৷ কিন্তু খুব সম্ভবতঃ তোমার মোবাইল তোমাকে প্রথমে ধমক দিয়ে বলবে-- 'এরকম অজানা অ্যাপ বিপজ্জনক হতে পারে, তাই install করব না৷'' তখন তোমাকে settings-এ গিয়ে privacy and security-র মধ্যে গিয়ে autoblock বন্ধ করে দিতে হবে৷ তারপর ডাউনলোড করা apk ফাইলটায় tap করলে install হবে৷ তারপর ফের settings-এ গিয়ে autoblock-টা ফের চালু করে দিও, নইলে আবার কোথা থেকে কোন্ দুষ্টু অ্যাপ ঢুকে পড়বে কে জানে!
