A curious fact about independence

We are going to discuss a paradox here. A paradox is born when two intuitive notions contradict each other. Let us start with discussing the contending intuitions for our paradox.

Intuition 1: Approximately same distribution

Suppose that we have two random variables $X,Y$. Then their joint CDF is $F(X\leq x,Y\leq y).$ This CDF characterises the joint distribution of $(X,Y)$, i.e., if we have two other random variables $S,T$ with same joint CDF, then $(X,Y)$ and $(S,T)$ has the same joint distribution. So it is only natrual to expect that if $(X,Y)$ and $(S,T)$ have approximately same joint CDF, then their distributions will also be approximately the same. By "approximately same" we mean for some very small $\epsilon$ we have $$\forall a,b\in{\mathbb R}~~|F_{XY}(a,b)-F_{ST}(a,b)|<\epsilon.$$ Here $F_{XY}$ and $F_{ST}$ are the joint CDFs of $(X,Y)$ and $(S,T),$ respectively. This belief is what powers statistics, where we really do not handle the random variables directly, but only through their observed values. If we have a large number of observatons of a random variable, all that we can hope to be able to get from them is an approximation of the underlying distribution. So if $F_{XY}\approx F_{ST}$, then there is no way you can distinguish them from data. In other words, data from $(X,Y)$ can be mistaken for that from $(S,T)$ and vice versa.

Independent and completely dependent

We know what is meant by two random variables being independent. In terms of CDF this means $$\forall x,y\in{\mathbb R}~~F_{XY}(x,y) = F_X(x)F_Y(y).$$ We shall abbreviate this to $F_{XY}\stackrel{\epsilon}{\approx}F_{ST}.$

Intuitively, this means knowledge about one of the variables sheds no light on the value of the other. What is the opposite extreme? Knowing one variable completely specifies the other variable: one is a deterministic function of the other.

The paradox

Given two independent random variables $X,Y$ and some $\epsilon>0$ we can manufacture two random variables $S,T$ such that $F_{XY}\stackrel{\epsilon}{\approx}F_{ST}$ and yet $S$ and $T$ are deterministic bijective functions of each other!!

More specifically, we have the following theorem.
Theorem Let $X,Y$ be independent random variables with density $f(x) = \left\{\begin{array}{ll}1&\text{if }0 < x < 1\\ 0&\text{otherwise.}\end{array}\right.$ Let $\epsilon>0.$ Then we can find random variables $S,T$ such that
  1. $F_S = F_X$ and $F_T = F_Y$
  2. $F_{XY}\stackrel{\epsilon}{\approx}F_{ST}$
  3. $S$ and $T$ are deterministic functions of each other.
In plain words, this means: If you choose $\epsilon$ to be the smallest positive number your computer can hold (e.g. $2^{-31},$ then no program running in that computer will be able to distinguish data from the perfectly dependent variables $S,T$ from data from independent variables, $X,Y$!!!

This surprising theorem has a constructive proof. Indeed, you can write a simple program to actually manufacture data from $(S,T)$ to see the counter-intuitive behaviour for yourself! The project is to explore the proof.

By the way, the theorem can be easily generalised to work with any continuous distributions for $X$ and $Y.$

The proof

We shall present the idea behind the proof step by step. We start with $S=X$ and $T=S.$ Then we shall make $T$ more and more complicated (but always piecewise linear!) functions of $S.$
