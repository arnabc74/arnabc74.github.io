Absolutely continuous distribution

We already know this definition:
Definition: A random variable $X$ is said to be continuous or to have a continuous distribution if its CDF is continuous.
A special class of such random variables is given in the following definition:
Definition: A random variable $X$ is said to be absolutely continuous or to have an absolutely continuous distribution if there is some $f:{\mathbb R}\rightarrow[0,\infty)$ such that for every $a\leq b\in{\mathbb R}$ we have $$P(X\in(a,b]) = \int_a^b f(x)\, dx.$$ Any such $f$ is called a probability density function (PDF) of $X.$

The following facts all come from continuity of probability.
Theorem Let $X$ have a density $f(x).$ Then
  1. $\forall a\in{\mathbb R}~~P(X=a) = 0.$
  2. $\forall a<b\in{\mathbb R}~~P(a < X < b) =P(a < X \leq b) =P(a \leq X < b) =P(a \leq X \leq b) = \int_a^b f(x)\, dx.$
  3. $\forall a\in{\mathbb R}~~P(a < X ) = P(a\leq X)=\int_a^\infty f(x)\, dx.$
  4. $\forall a\in{\mathbb R}~~P(X < a) = P(X\leq a)=\int_ \infty^a f(x)\, dx.$
  5. $\int_ \infty^ \infty f(x)\, dx = 1.$

This last condition is particularly useful.

Theorem If we have some $f:{\mathbb R}\rightarrow[0,\infty)$ with $\int_ {-\infty} ^\infty f(x)\, dx = 1,$ then the function $F:{\mathbb R}\rightarrow{\mathbb R}$ defined as $$F(x) =\int_{-\infty}^x f(t)\, dt$$ is a CDF.

Hence by the fundamental theorem of probability we have the following corollary:
Theorem A function $f:{\mathbb R}\rightarrow[0,\infty)$ is a density of some random variable if and only if with $\int_ {-\infty} ^\infty f(x)\, dx = 1.$
It is a somewhat disconcerting fact that density of an absolutely continuous distribution is not unique. For intance, if changing at any countable numbr of points would still keep it a density (assuming non-negativity). So there cannot be any way to recover the density given the CDF. However, the following result provides some relief.

Theorem If $f(x)$ and $g(x)$ are both densities for the same distribution and both are continuous at some $a\in{\mathbb R},$ then $f(a)=g(a).$

Proof: Let, if possible, $f(a)\neq g(a).$ Say $f(a)> g(a).$

Let $h(x) = f(x)-g(x).$ Then $h(x)$ is continuous at $x=a$ and $h(a)>0.$

So $\exists\delta>0~~\forall x\in (a-\delta,a+\delta)~~h(x)>\frac{h(a)}{2}.$

Hence $\int_{a-\delta}^{a+\delta} h(x)\, dx \geq \delta h(a) > 0.$

But this is impossible, since $\int_{a-\delta}^{a+\delta} f(x)\, dx =\int_{a-\delta}^{a+\delta} g(x)\, dx, $ since both are equal to $P(a-\delta< X < a + \delta).$ [QED]

It is possible to recover a density from the CDF as follows. Let $F$ be the CDF. Then define $f$ as folllows; $$f(x) = \left\{\begin{array}{ll}F'(x)&\text{if }\mbox{it exists}\\ 0&\text{otherwise.}\end{array}\right. $$ Of course, if the distribution is not absolutely continuous, then this may lead to meaningless reult. For example, for a discrete distribution, the CDF is a step function, and so the above prescription would lead to $f(x)\equiv 0.$ It is possible to characterise CDFs arising out of densities.

We had already defined expectation of a random variable in general. That definition reduces to the following:
Theorem Let $X$ be a random variable with density $f.$ Let $h:{\mathbb R}\rightarrow[0,\infty)$ be a any function such that $h(X)$ is a random variable. Then $E(h(X))=\int_{-\infty}^\infty h(x) f(x)\, dx.$ This may be $\infty.$

If $h:{\mathbb R}\rightarrow{\mathbb R}$, then $E(h(X)) = E((h(X))_+) - E((h(X))_-)$, as dictated by the general definition of expectation. This may be finite, $\infty$, $-\infty$ or may not exist.

Since this is a special case of the general definition, all the properties (like linearity, monotonicity) are intact.

We define $V(X)$ and the other moments as usual. Their properties (e.g., Chebyshev inequality) are intact.
