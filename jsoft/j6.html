/ (monad) If u is a verb, then u/ (monad) is "continued u".
/ (dyad) If u is a verb, then u/ (dyad) is outer product.
&. (dyad adverb) Conjugation: u&.v is $v ^{-1} u v$ as monad. (As a dyad?)
& (dyad adverb) 3&u (as a monad) means the function $y\mapsto u(3,y)$. Similarly, u&3 (as a monad) means the function $x\mapsto u(x,3)$.
@ (dyad adverb) u@v (as a monad) means the function $y\mapsto u(3,y)$. Similarly, u&3 (as a monad) means the function $x\mapsto u(x,3)$.
1!:2 Writes into a file. fw =: 1!:2 and then 'string to write' fw <'filename' Notice that this writes a string. The writing is a one-shot activity including file open, write and file close. If you want to write many things into a file, first accumulate them into a single string (insert newlines appropriately), and then use this once. Overwrites any existing file of the same name.
/: (dyad and monad) x /: y sort x according to increasing order of y (ie, finds the permutation that sorts y in increasing order, and applies that permutation to x). It is also a nice way to permute a list. Just supply the permutation as y. Also x /: x (or /:~ x) up-sorts x.

A monad it gives the permutation that up-sorts x.
\: (dyad and monad) Similar to /: but in the opposite direction.
": (dyad and monad) Converts numbers to strings.
< and > (monad) Boxing and unboxing.
