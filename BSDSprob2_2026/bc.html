1. Infinitely often

Suppose that we have a sequence of subsets of some universal set. We shall define a new set from this sequence as follows.
Definition: Infinitely often (i.o.) Let $(A_n)$ be a sequence of subsets of some universal set. Then we define $$\{A_n\io\} = \cap_n \cup_{k\geq n} A_k.$$

EXAMPLE 1:  Suppose that $A_n = \left[ -\frac 1n, 1-\frac 1n \right].$ Find $\{A_n\io\}.$

SOLUTION: You can think of this sequence as $[-1,0]$ travelling towards $[0,1].$

Here $\cup_{k\geq n} A_k = \left[ -\frac 1n, 1\right).$

So $\{A_n\io\} = \cap_n \cup_{k\geq n} A_k = \cap_n [0, 1).$ ■

The definition may appear rather complicated. Let us digest itintuitively:
  • Consider all the elements which are in all the $A_n$ s. The set of all such elements is $\cup_{k\geq 1} A_k. $
  • Next consider all the elements which are in all the $A_n$ s except possibly $A_1.$ All the earlier elements are already there, plus possibly some extra. The set of all these elements is $\cup_{k\geq 2} A_k. $
  • Similarly, $\cup_{k\geq 3} A_k $ is the set of all elements that are in all the $A_n$ s except possible the first two.
If we take the intersection of all such unions of the form $\cup_{k\geq n} A_k,$ we get $\{A_n\io\}.$ Thus $\liminf A_n$ is the set all elements that are in all the $A_n$'s except for a finitely many, i.e., all the lements are are infinitely many of the $A_n\$'s. Hence the name infinite often.

The following result often proves handy when proving $P(A_n)\rightarrow 0.$

TheoremLet $(A_n)$ be a sequence of events with $P(A_n\io) = 0.$ Then we must have $P(A_n)\rightarrow0.$

A proof of this theorem is outlined in the exercises below.

2. Borel-Cantelli lemmas

First Borel-Cantelli lemma Let $(A_n)$ be a sequence of events in some probability space. If $\sum P(A_n) < \infty,$ then $P(A_n\io)=0.$

Proof: $$\begin{eqnarray*} P(A_n\io) & = & P(\cap_n\cup_{k\geq n} A_k)\\ & = & P( \lim_n\cup_{k\geq n} A_k)\\ & = & \lim_n P(\cup_{k\geq n} A_k). \end{eqnarray*}$$ Now $P(\cup_{k\geq n} A_k)\leq \sum_{k\geq n} P(A_k).$

Since $\sum P(A_n) < \infty$, hence $\lim_n \sum_{k\geq n} P(A_k) = 0.$

Hence the result. [QED]

Second Borel-Cantelli lemma Let $(A_n)$ be a sequence of independent events in some probability space. If $\sum P(A_n)= \infty,$ then $P(A_n\io)=1.$

Proof:Skipped.[QED]

The second lemma makes the assumption of independence. The following counterexample shows what may happen if this assumption were dropped.

EXAMPLE 2:  Let $U\sim Unif(0,1)$ and for $n\in{\mathbb N}$ let $$A_n = \left\{U\in\left(0,\frac 1n\right)\right\}.$$ Notice how we are defining all the $A_n$'s based on the same $U.$ Clearly, $A_n$'s are not independent. For instance, $P(A_n)=\frac 1n$, but $P(A_n|A_{n+1})=1.$

Here $\sum_n P(A_n) = \sum_n \frac 1n = \infty,$ but $P(A_n \io) = 0.$ ■

3. Demonstrations

3.1. Monkey typing Hamlet

A common demonstration of the second Borel-Canteli lemma is show that if a monkey types a keyboard randomly, then it will type the exact text of Hamlet infinitely often.

Let the exact text of Hamlet have $h$ letters. Consider packets of $h$ letters. Let $A_n$ be the event that the $n$-th packet is an exact copy of Hamlet. Then $A_n$'s are independent and $P(A_n)>0.$ So $\sum_n P(A_n) = \infty.$ Hence by the second Borel-Canteli lemma, $P(A_n\io)=1.$

EXERCISE 18: However, if you want to just show that the monkey will surely write Hamlet at least once, then you really do not need to use the second Borel Cantelli lemma. You can prove it much more easily using continuity of probability. How?

3.2. Project: Normal numbers

Consider a uniform random number between 0 and 1. Let $d_n$ be the $n$-th digit after the decimal point in its (unique) non-terminating decimal expansion. For example, if the number is $\frac 12,$ then the non-terminating decimal expansion is $0.499999...,$ and so $d_1 = 4,$ while $d_n=9$ for each $n\geq 2.$

Consider the event $\{d_1=3\}.$ Clearly it is $(0.3,0.4].$ So this has probability $\frac{1}{10}.$ Next consider the event $\{d_2=3\}.$ This is $\cup_{d_1=0}^9 \frac{d_1}{10}+(0.03,0.04].$ Again, the probability is $\frac{1}{10}.$ In particular we have the following theorem:

Theorem
  • For each $n$ the random variable $d_n$ is uniformly distributed over $\{0,1,...,9\}.$
  • If $m\neq n,$ then $d_m$ and $d_n$ are independent.

Next we shall fix a single number $\omega\in(0,1)$ and check how many times any given digit (say 3) occurs in it.

Let $S_n(\omega) = $ number of times the digit 3 occurs among the first $n$ digits of $\omega.$ For instance of $\omega=\frac 13,$ then $S_n(\omega) = n.$

We are interested in the proportion of times a digit occurs in the expansion. Mathematically, we work with $\lim_{n\rightarrow \infty} \frac{S_n(\omega)}{n}$ (which may not exist). For instance, if $\omega=\frac 13,$ then the limit exists and equals $1.$ If $\omega=0.3,$ then the limit exists and equals $0.$

3.3. Project: Random walk in 3D

Let $S = X_1+\cdots+X_n$ a simple, symmetric random walk in 3D. We shall show that $P(S_n=0\io)=0.$ Thanks to the first Borel Cantelli lemma, it is enough to show that $\sum P(S_n=0) < \infty.$

Now, $P(S_n=0)=0$ if $n$ is odd. So let us take $n=2(a+b+c),$ where $a$ is the number of $+x$ steps (and $-x$ steps). Similarly, for $b,c.$

Total number of paths of length $2k$ is $6^{2k}.$ Number of favourable paths is $$ \sum_{a,b,c:a+b+c=k} \frac{(2k)!}{a!^2 b!^2 c!^2}. $$ By convexity of $\log m!$, the maximum term occurs when $a,b,c$ are as equal as possible. Hence $T_n\leq C n^2/((n/3)!)^6.$

Using Stirling's approximation $m!\sim \sqrt{2\pi m}(m/e)^m,$ we have $P(S_{2k}=0)\leq C n^{-3/2}.$

Hence the result.
