Almost sure and probability convergences

Definition: Amost sure convergence Let $(X_n),X$ be random variables all defined on some common probability space $(\Omega,\calF, P).$ Then we say that $(X_n)$ converges to $X$ almost surely and write $X_n\rightarrowA X$ if $P(X_n\rightarrow X) = 1.$

EXAMPLE 1: Let our probability space be $Unif(0,1)$ with $\Omega = {\mathbb R}.$ Let $X_n(\omega) =\omega^n$, and let $X(\omega)\equiv 0.$ Then $X_n\rightarrowA X.$

However, if we replace $Unif(0,1)$ with $Unif(0,2),$ then the convergence breaks down. ■

Definition: Probability convergence Let $(X_n)$ be a sequence of random variables all defined on some common probability space $\Omega,\calF, P).$ Let $X$ be some random variable also defined on the same probability space. Then we say that $(X_n)$ converges to $X$ in probability and write "$X_n\rightarrowP X$ if $$\forall \epsilon>0~~P(|X_n-X|> \epsilon)\rightarrow 0.$$ It is also called weak convergence.
The condition that all the $X_n$'s should be defined on a common probability space may be relaxed if $X$ is a degenerate random variable.

Infinitely often

Definition: Infinitely often Let $(\Omega,\calF,P)$ be any probability space. Let $(A_n)$ be a sequence of events in it. The event "$A_n$'s occur infinite often" (abbreviated to "$A_n\io$") is defined as $\{\omega\in\Omega~:~\omega \in A_n$ for infintely many $A_n\mbox{'s}\}.$

We can express it as $\cap_{n}\cup_{k\geq n} A_k.$

Borel-Cantelli lemmas

First Borel-Cantelli lemma Let $(A_n)$ be a sequence of events in some probability space. If $\sum P(A_n) < \infty,$ then $P(A_n\io)=0.$

Proof: $$\begin{eqnarray*} P(A_n\io) & = & P(\cap_n\cup_{k\geq n} A_k)\\ & = & P( \lim_n\cup_{k\geq n} A_k)\\ & = & \lim_n P(\cup_{k\geq n} A_k). \end{eqnarray*}$$ Now $P(\cup_{k\geq n} A_k)\leq \sum_{k\geq n} P(A_k).$

Since $\sum P(A_n) < \infty$, hence $\lim_n \sum_{k\geq n} P(A_k) = 0.$

Hence the result. [QED]

Second Borel-Cantelli lemma Let $(A_n)$ be a sequence of independent events in some probability space. If $\sum P(A_n)= \infty,$ then $P(A_n\io)=1.$

Proof:Skipped.[QED]

The second lemma makes the assumption of independence. The following counterexample shows what may happen if this assumption were dropped.

EXAMPLE 2:  Let $U\sim Unif(0,1)$ and for $n\in{\mathbb N}$ let $$A_n = \left\{U\in\left(0,\frac 1n\right)\right\}.$$ Notice how we are defining all the $A_n$'s based on the same $U.$ Clearly, $A_n$'s are not independent. For instance, $P(A_n)=\frac 1n$, but $P(A_n|A_{n+1})=1.$

Here $\sum_n P(A_n) = \sum_n \frac 1n = \infty,$ but $P(A_n \io) = 0.$ ■

Demonstrations

Monkey typing Hamlet

A common demonstration of the second Borel-Canteli lemma is show that if a monkey types a keyboard randomly, then it will type the exact text of Hamlet infinitely often.

Let the exact text of Hamlet have $h$ letters. Consider packets of $h$ letters. Let $A_n$ be the event that the $n$-th packet is an exact copy of Hamlet. Then $A_n$'s are independent and $P(A_n)>0.$ So $\sum_n P(A_n) = \infty.$ Hence by the second Borel-Canteli lemma, $P(A_n\io)=1.$

However, you really do not need to use the second Borel Cantelli lemma here. It also follows from continuity of probability.

Normal numbers

Consider a uniform random number between 0 and 1. Let $d_n$ be the $n$-th digit after the decimal point in its (unique) non-terminating decimal expansion. Then $\forall k\in\{0,1,2,...,9\}~~P(d_n=k\io) = 1.$ So, in particular, $P(0\mbox{ occurs }\io)=1.$ Hence, $P(0,1\mbox{ occur }\io)=1.$ Proceeding similarly, $P(\mbox{each digit occurs }\io)=1.$

Random walk in 3D

Let $S = X_1+\cdots+X_n$ a simple, symmetric random walk in 3D. We shall show that $P(S_n=0\io)=0.$ Thanks to the first Borel Cantelli lemma, it is enough to show that $\sum P(S_n=0) < \infty.$

Now, $P(S_n=0)=0$ if $n$ is odd. So let us take $n=2(a+b+c),$ where $a$ is the number of $+x$ steps (and $-x$ steps). Similarly, for $b,c.$

Total number of paths of length $2k$ is $6^{2k}.$ Number of favourable paths is $$ \sum_{a,b,c:a+b+c=k} \frac{(2k)!}{a!^2 b!^2 c!^2}. $$ By convexity of $\log m!$, the maximum term occurs when $a,b,c$ are as equal as possible. Hence $T_n\leq C n^2/((n/3)!)^6.$

Using Stirling's approximation $m!\sim \sqrt{2\pi m}(m/e)^m,$ we have $P(S_{2k}=0)\leq C n^{-3/2}.$

Hence the result.

$L_p$ convergence for $p\geq1$

Definition: $L_p$ convergence Let $p\geq 1.$ Let $(X_n), X$ be random variables defined on some probability space. We say that $(X_n)$ converges to $X$ in $p$-th moment or in $L_p$ if $E\big(|X_n-X|^p\big)\rightarrow 0.$

We write this as $X_n\rightarrowL p X.$

Theorem Let $p > q \geq 1.$ Then if $X_n\rightarrowL p X$ we must also have $X_n\rightarrowL q X.$

Proof: Let $Y_n = X_n-X.$

Take $r=\frac pq$ and $s=\frac{p}{p-q}$ so that $\frac 1r+\frac 1s = 1.$

We have $|Y_n|^q = |Y_n|^q \cdot 1.$

Then by Holder inequality $$ (\|Y_n\|_q)^q \leq (\|Y_n^q\|_r)^q \|1\|_s. $$ Since $qr = p,$ $$ (\|Y_n\|_q)^q \leq (\|Y_n\|_p)^q $$ Hence the result. [QED]

EXERCISE 1:  If $X_n\rightarrow L 1 X,$ then $E(X_n)\rightarrow E(X),$ but the converse is not true.

[Hint]

Take $X_n\equiv (-1)^n$ and $X\equiv 1$ as a counterexample for the converse.

The following example shows that almost sure convergence does not imply $L_p$ convergence, even when the $p$-th moments exist finitely for all the random variables involved.

EXAMPLE 3: We work with $Unif(0,1)$ probability space. Let $X_n(\omega) = \left\{\begin{array}{ll}n&\text{if }\omega\in\left(0,\frac 1n\right)\\ 0&\text{otherwise.}\end{array}\right.$.

Let $X\equiv 0.$

Then clearly $\forall\omega\in (0,1)~~X_n(\omega)\rightarrow X(\omega).$ Hence $X_n \rightarrowA X.$

But $\forall n\in{\mathbb N}~~E(X_n) = n\times\frac 1n = 1\not\rightarrow 0 = E(X).$ ■
