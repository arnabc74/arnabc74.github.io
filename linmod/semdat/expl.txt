name: a string representing the name of the video file
extn: a string representing the extension of the video file
       (e.g., if the file is called abc.xyz, then name is "abc"
       and extn is "xyz")
hour, min, sec: the duration of the video.
      (e.g., if the video is 123 min 34 sec long, then hour=2,
      min=3, sec=34)
size: size of the video file in bytes.
date: date of creation of the video file.
