Sequential tests

Computing error probabilities

A SPRT involves two numbers A and B. We shall compute the probabilities of type I and type II errors in terms of these. This will help us to choose A and B in a way so that the resulting test has specified error probabilities.

In the following discussion we shall use the notation P0(.) to denote probabilities under H0, and similarly P1 for probabilities under H1. Also let Sm denote the sum of Z1,...,Zm.

(*) and (**) are called Wald's inequalities.

If you treat these as equalities then you get Wald's approximation:

A = (1-β)/α
B = β/(1-α).
Of course, these are not exact. But Wald suggested using these to compute A and B from specified α and β. Here is the basic argument that Wald provided for this approximation.

Notice first that if we could change the test to

Accept if LR = B, and reject if LR=A,
then Wald's approximation would be exact. The only problem with these is that the LR may very well jump out of [B,A] without actually taking the values B or A. In other words, ST may be below log(B) or above log(A). This is called excess over boundaries. The excess is never more than one Zi. Hopefully, the Zi's take large values with small probabilities, and hence the excess over boundaries will be small with large probabilities. This is all heuristics. Let us explore it using computer simulation. Here we want to relate the error incurred due to Wald's approximation with the excess over boundaries.

Exercise SPRT2.6: Wald's approximation affects the error probabilities conservatively. Show this as follows. Suppose we want a SPRT with error probabilities α and β. We apply Wald's approximation to get A, B. Since we did an approximation, the actual error probabilities (call them α' and β', say) need not equal α and β. Show that
α' + β' α + β.
We would be happier if we could say something like

α' α and β' β.
Indeed, Wald had some intuitive argument to suggest that this is so. The following exercise explores this using simulation.

Exercise SPRT2.7: Take different values of α and β. Compute A, B using Wald's approximation. Obtain α' and β' by simulation. Plot α' against α, and β' against β.

Expected sample size

A sequential statistical procedure needs random sample size. It is a natural question to ask what the expected sample size is. If T is the stopping rule used, then this is E(T). We shall show later how to find this for SPRT. We shall also show that SPRT minimises E(T) in the following sense.

Theorem: (Optimality of SPRT) Let φ be a SPRT with error probabilities α and β. Let its stopping rule be called T. Let ξ be any sequential test with stopping time T' such that
  • P(type I error) α
  • P(type II error) β
  • E(T') is finite under both the hypothesis.
Then
E(T) E(T').

Proof: We shall do later in this course.
