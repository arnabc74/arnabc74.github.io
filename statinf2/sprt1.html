Sequential tests

Introduction

Example: Suppose that we have data
X1,...,X10 iid N(μ,1).
We want to test
H0: μ =0 Vs. H1: μ = 5.
A typical test is of this form: Reject H0 if > 3.7, else accept H0.

Now suppose that = 3.701. So we should reject H0. But had been 3.699, we should have accepted H0. Is this sensible? Since, is based on only 10 observations from N(m,1), its variance is 0.1. So the difference between 3.699 and 3.701 is a minor one for . Yet the entire conclusion changed based on that.

This is a borderline case. Ideally one would like to say that the based on the present data the test in inconclusive. Rather than accept or reject H0 right now, we should better get some more data. This is the main idea behind sequential tests.

Definition: Suppose that we have data
X1,X2,...
A sequential test based on this data is a decision rule that tells for each n to take one of the following actions based on the observed values of
X1,...,Xn:
  1. accept H0
  2. reject H0
  3. get one more sample

    Equivalently, a sequential test is a stopping time, T, together with a decision rule telling us whether to accept/reject H0 based on

    X1,...,XT
    once we stop.

Sequential Probability Ratio Test (SPRT)

In classical statistics we have likelihood ratio tests. In such a test the test statistic is the ratio of the likelihood of the observed data under H0 and H1. If this ratio is large then we reject H0, else we accept H0. Neyman-Pearson lemma tells us that this is the optimal test for a given level.

It is customary to work with log-likelihood ratio than the likelihood ratio.

Suppose we have data

X1,,X2,... iid with density f.
We want to test
H0: f = f0 Vs H1: f = f1
Define
Zi = log( f1(Xi)/f0(Xi)).
Then the Zi's are iid, and the log-likelihood ratio is their sum.

Definition: In this set up a SPRT with cutoffs A, B (0 < B < 1 < A) (also written as SPRT(A,B)) is defined as the sequential test that goes on sampling until the likelihood ratio(LR) goes outside (B,A). Once it falls outside this interval, we stop sampling and accept H0 if the LR is below B, reject H0 if the LR is above A.

Theorem: (Stein's lemma) Under the condition
P(Z1 = 0) < 1 under both f0 and f1
the stopping time, T, of SPRT(A,B) is finite with probability 1 under both f0 and f1.
